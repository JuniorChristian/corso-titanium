1 Lezioni

1.1 28 Aprile Setup & get Running

1.1.1 Installazione

Per il corretto funzionamento è necessario installare:

  • Java JDK (OpenJDK 7?)
  • install nodejs da [http://nodejs.org/download/]

    Linux package 

    $ sudo apt-get install nodejs npm
  • Android SDK [http://developer.android.com/sdk/index.html]
    • installare pacchetti API
    • creare avd iniziale
  • Installare i pacchetti titanium con: 
    $ npm install -g titanium alloy acs
  • Titanium Studio da [http://www.appcelerator.com/titanium/download/]

1.1.2 Panoramica funzionalità

  • Per lo più utilizza componenti nativi di cui viene fornita una specifica API JavaScript; []
  • Per ragioni di portabilità del codice a volte astrae "concetti" simili nelle varie piattaforme con oggetti compositi;
